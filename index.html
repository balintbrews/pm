Kléri Bálint

Prezentáció

Rövid prezentáció a Page Managerről és a CTools pluginok működéséről, architektúrájáról, valamint ezek Panels modullal való kapcsolatáról.

Page Manager gyakorlat

Gyakorlat saját CTools pluginok készítéséhez

A feladat Page Manager modul segítségével létrehozni egy oldalt, amely a tweet/[tweet_id] oldalon az azonosító által meghatározott tweethez tartozó pár tetszőleges adatot megjelenít.

Minta modul

A Drupalaton Tweets modul bemutatja, miként lehet saját argument, context és content type CTools pluginokat készíteni, melyek segítségével Page Managerben egyszerűen megoldható a fent vázolt feladat.

Workshop gyakorlat

A workshopon résztvevőknek és kódoláshoz kedvet érzőknek a modul a váza elérhető a workshop branch-ben:

$ git clone --recursive --branch workshop http://git.drupal.org/sandbox/balint.kleri/1714096.git drupalaton_tweets

Kész megoldás

Akik a kész megoldásra, vagyis a teljes kódra kíváncsiak, azok számára a 7.x branch tartalma az érdekes:

$ git clone --recursive --branch 7.x http://git.drupal.org/sandbox/balint.kleri/1714096.git drupalaton_tweets

Készült a GitHub Pages szolgáltatás és a Dinky smink segítségével.
