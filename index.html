Page Manager gyakorlat

A feladat egy egyszerű projekt management rendszer kialakítása.

User storyk

  • Adminisztrátorként szeretnék projekteket létrehozni.
  • A csapat tagjaként szeretnék feladatokat létrehozni és jelezni, hogy azok mely projekthez kapcsolódnak.
  • Adminisztrátorként szeretnék csapattársakat projektekhez rendelni.
  • A csapat tagjaként szeretném látni a projektek listáját.
  • A csapat tagjaként szeretném egy projekt saját oldalán látni a projekthez tartozó feladatok listáját táblázatos formában.
  • A csapat tagjaként szeretném egy feladat saját oldalán látni a többi feladatot, amely ugyanahhoz a projekthez tartozik, mint a jelenleg megtekintett feladat.
  • A csapat tagjaként szeretném egy feladat saját oldalán látni a többi csapattárs fotóját, akik ugyanazon a projekten dolgoznak, amihez az éppen megtekintett feladat kapcsolódik.

A felhasználói jogosultságok konfigurálását a gyakorlat nem érinti. A cél egy nagyon egyszerű megoldás megvalósítása csakis core modulok, Page Manager, Panels és Views használatával, és mindenféle kódírás nélkülözésével.

Gyakorlat saját CTools pluginok készítéséhez

A feladat Page Manager modul segítségével létrehozni egy oldalt, amely a tweet/[tweet_id] oldalon az azonosító által meghatározott tweethez tartozó pár tetszőleges adatot megjelenít.

Minta modul

A Drupalaton Tweets modul bemutatja, miként lehet saját argument, context és content type CTools pluginokat készíteni, melyek segítségével Page Managerben egyszerűen megoldható a fent vázolt feladat.

Workshop gyakorlat

A workshopon résztvevőknek és kódoláshoz kedvet érzőknek a modul a váza elérhető a workshop branch-ben:

$ git clone --branch workshop http://git.drupal.org/sandbox/balint.kleri/1714096.git drupalaton_tweets

Kész megoldás

Akik a kész megoldásra, vagyis a teljes kódra kíváncsiak, azok számára a 7.x branch tartalma az érdekes:

$ git clone --branch 7.x http://git.drupal.org/sandbox/balint.kleri/1714096.git drupalaton_tweets

Készült a GitHub Pages szolgáltatás és a Dinky smink segítségével.
